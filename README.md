# NextAI
A Web-based Full-stack question-answering AI agent for Querying PDFs
Tech Stack：Node.js, Express, React, Ant Design, LangChain, OpenAI GPT-3.5 Turbo, Embedding, Vector Storage, RESTful API
●	We created an interactive conversational UI that leverages React and Ant Design to enable users to easily upload and interact with PDF documents in real time.
●	Implemented high-performance RESTful APIs using Node.js and Express.
●	Memory vectors are used to store cache-generated embeddings for efficient retrieval.
●	OpenAI's GPT-3.5 Turbo API and Langchain technology are integrated to develop a high-level AI agent capable of loading, splitting, storing, retrieving and outputting documents.
